## Overview

A powerful tool that transforms simple Markdown files into stunning, production-ready portfolio websites. Perfect for developers who want to showcase their work without touching HTML/CSS.

## Features

- **Zero Configuration**: Just write Markdown and let the generator handle the rest
- **Custom Themes**: Choose from 10+ pre-built themes or create your own
- **Automatic Deployment**: Push to GitHub and get instant preview
- **SEO Optimized**: Built-in meta tags and sitemap generation
- **Dark Mode**: Automatic dark/light theme detection

## Quick Start

```bash
npm install -g terminal-portfolio
terminal-portfolio init my-portfolio
terminal-portfolio build
terminal-portfolio preview
```

## Why Build This?

I created this tool because I was tired of spending more time designing portfolios than actually building projects. Now I can focus on what matters - coding great software.

## Tech Stack

Built with Jekyll and custom Liquid filters. The entire generator is under 500 lines of Ruby code.
