## Overview

TaskFlow is a modern task management application that helps you stay organized without the clutter. Built with React Native for seamless cross-platform experience.

## Features

- **Smart Lists**: AI-powered task organization and prioritization
- **Offline First**: Full functionality without internet connection
- **Sync Across Devices**: Real-time sync via cloud backend
- **Natural Language Input**: Type "Meeting tomorrow at 3pm" and watch it happen
- **Focus Mode**: Distraction-free task entry and completion
- **Beautiful Animations**: Smooth, native animations throughout

## Screenshots

The app features a clean, minimalist design with smooth animations and haptic feedback. Dark mode support included.

## Tech Stack

- **Frontend**: React Native with TypeScript
- **State Management**: Redux Toolkit
- **Local Database**: WatermelonDB for offline support
- **Sync**: Custom WebSocket-based sync engine

## Why I Built It

After trying dozens of task managers, I couldn't find one that balanced power with simplicity. TaskFlow is the app I wished existed.
